# Restaurant Website

A modern and responsive Restaurant Website built using HTML, CSS, and JavaScript. This project provides an interactive platform for customers to explore the restaurant's offerings, view menu items, and make reservations through a user-friendly interface.

## Features

- Home Page with attractive welcome section
- Menu Page with categorized food items and prices
- About Us page with restaurant information
- Reservation Form for customer bookings
- Contact Page
- Customer Reviews section
- Chef's Special dishes
- Food categories including Starters, Main Course, Desserts, Beverages, and Mocktails
- Responsive and visually appealing design
- Interactive functionality using JavaScript

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/yourusername/Restaurant-Website.git
```

2. Open the project folder.

3. Launch `index.html` in your browser.
